# Wortmeister

Bu wortmeister'in yeni sürümüdür. Uygulama her platform için erişilebilir olması için web tabanlı olarak yeniden tasarlandı. Render ile deploy edildi ve artık kullanıma hazır.

## Yenilikler

- Uygulama tasarımı modern tarzda yeniden tasarlandı
- Cevap kontrol mekanizması geliştirildi. Artık program yazım hatalarına karşı daha duyarlı.
- Uygulamayı kolaylaştırmak adına ipucu seçeneği eklendi.
- Animasyonlar eklenerek uygulama daha çekici hale getirildi.
- Rust 🦀 backend'i kullanılarak uygulama hem daha hızlı hem de stabil hale getirildi.
- Tüm kullanıcılar için json dosyaları oluşturularak ilerlemeleri kaydedildi.
